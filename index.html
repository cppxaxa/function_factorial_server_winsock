Winsock Server for responding factorial

A sever implemented in winsock2.h that responds by giving factorial of a number sent to it

Usage

Open up the Visual Studio Make a console program with the contents Run the program

Instruction

You can use Telnet client, start the Telnet client write "open 127.0.0.1 8888" ( Replace 127.0.0.1 with the computer ip running the program ) write "5=" The server will respond by giving result "120"
